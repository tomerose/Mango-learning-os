# MangoOS — 投资人一页摘要

**日期：** 2026年6月 | **阶段：** MVP 验证

---

## 项目一句话

**MangoOS = 面向学生的 AI 学习与任务成果操作系统。** 将课程、资料和目标转化为可交付、可导出、可复用的结构化学习成果（PDF 讲义、复习包、闪卡、学习计划）。

---

## 用户痛点

- AI 聊天工具泛滥，但学生**无法将它们转化为可交付成果**
- 期末复习、论文整理、资料归档等高频刚需场景**缺少专用 AI 工具**
- 学习过程碎片化：聊天、笔记、题库、文档工具各自独立，**没有闭环**
- 学生想"用 AI 做点什么"，但不知道怎么做、做不出成品

---

## 产品方案

**输入→检索→生成→审核→导出→沉淀** 六步闭环：

1. 用户输入学习任务（"生成微观经济学期末复习讲义"）
2. MangoAgent 自动联网检索 11 个数据源
3. DeepSeek AI 生成结构化内容（18 章节）
4. Quality Gate v4 自动审核（7 个维度，Pro 用户 90 分硬门禁）
5. 导出为 PDF / Word / Markdown / HTML
6. 结果保存到 Library，可复用、可版本追溯

---

## 核心优势

| 维度 | 优势 |
|------|------|
| **成果导向** | 不是聊天机器人，是学习成果生产系统 |

| **场景真实** | 创始人（大一学生）直接理解目标用户 |
| **极低成本** | 月度运营 < ¥400，单人全栈开发 |
| **快速迭代** | V1.0→V14.8.1 数周完成，周级更新 |
