## 下一步操作 (用户)

### 选项 A: 继续演示模式
无需配置，打开 `http://localhost:3003` → 点击 **"以游客身份继续"**：
- 核心交互全部可用 (任务、笔记、闪卡、反思)
- 数据存 localStorage
- AI 功能返回演示文本

### 选项 B: 启用完整功能
编辑 `.env.local`：

```bash
# 1. Supabase (云端数据)
# 注册 https://supabase.com → 创建项目 → Settings → API
NEXT_PUBLIC_SUPABASE_URL=https://你的项目.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=你的_anon_key
SUPABASE_SERVICE_ROLE_KEY=你的_service_role_key

# 2. DeepSeek (AI 功能)
# 注册 https://platform.deepseek.com → 创建 API Key
AI_API_KEY=sk-你的密钥
AI_BASE_URL=https://api.deepseek.com
AI_MODEL=deepseek-chat
```

配置后：
1. 在 Supabase SQL Editor 运行 `docs/architecture/database-schema.sql`
2. 重启开发服务器: `npm run dev`
3. 注册账号即可使用云端 + AI 完整功能
