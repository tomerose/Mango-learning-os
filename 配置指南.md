# AI Learning OS - 环境变量配置指南

## 问题诊断

### Vercel CLI 限制
Vercel 加密的环境变量无法通过 `vercel env pull` 拉取到本地（安全机制）。
所有敏感变量在本地显示为空字符串 `""`。

---

## 解决方案：手动配置本地环境

### 1. 从 Vercel Dashboard 获取密钥

访问：https://vercel.com/mango-s-projects5/mango-learning-os/settings/environment-variables

复制以下变量的值：
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`
- `AI_API_KEY`
- `AI_BASE_URL`
- `AI_MODEL`

### 2. 编辑本地 `.env.local`

```bash
# D:\Claudecoda学习\AI-Learning-OS\.env.local

# ─── Supabase (云端数据) ───────────────────────────
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxx...
SUPABASE_SERVICE_ROLE_KEY=eyJxxx...

# ─── DeepSeek (AI 功能) ────────────────────────────
AI_API_KEY=sk-xxx...
AI_BASE_URL=https://api.deepseek.com
AI_MODEL=deepseek-chat
```

### 3. 重启开发服务器

```bash
cd D:\Claudecoda学习\AI-Learning-OS
npm run dev
```
